Turtle Road by Daniel Murillo 

2022 



Hello! 


In order to run this game please make sure you are running the latest version of Python3 on your PC or MAC. 


To run:

-Open the turtleroad folder and run the main.py file using python in your PC's terminal. 


MAC:
$ ~ python3 main.py


Windows:
C:/ python main.py


This is a "frogger-style" game using Python's built-in Turtle module. 
The object is to get the turtle across the road using the "UP" arrow key on your keyboard.
Make sure not to hit a "car" or your game will be over. 
TRY AND BEAT THE HIGH SCORE!!
